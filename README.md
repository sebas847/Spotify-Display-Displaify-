# Spotify-Display-Displaify-
The project involves the design and development of an IoT (Internet of Things) device based on the ESP32-C3 microcontroller, capable of wirelessly connecting to the Spotify streaming platform. The main objective is to display, in real time, information about the music the user is listening to (song title and artist name) on an ST7735 TFT screen.
